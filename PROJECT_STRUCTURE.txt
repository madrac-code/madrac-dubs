CORE MODULES SUMMARY
====================

Pipeline (8 stages):
  1. Validate inputs (10%)
  2. Extract audio from video (25%)
  3. Generate TTS for subtitles (50%)
  4. Reduce original vocals (60%)
  5. Synchronize TTS to subtitle timing (70%)
  6. Normalize loudness to LUFS (75%)
  7. Mix reduced audio with dubbed TTS (80%)
  8. Export to MKV with muxed audio (100%)

TTS Engine:
  - Abstract TTSEngine interface
  - Edge TTS implementation (200+ voices, 50+ languages)
  - Support for Spanish, English, French, Portuguese, Italian, German, Japanese, Chinese, Russian, Arabic

Audio Processing:
  - Vocal reduction (center channel cancellation + EQ)
  - TTS synchronization (time-stretch to subtitle duration)
  - Loudness normalization (broadcast-standard LUFS)
  - Audio mixing (blend original + dubbed)

CLI & API:
  - Command-line interface with click
  - Flask HTTP API server (localhost:5000)
  - Job submission and status polling
  - Progress tracking (0-100%)

INTEGRATION WITH MADRAC-SUBS
=============================

Zero modifications to madrac-subs core!

Integration methods:
  1. HTTP API: Submit jobs, poll status
  2. File exchange: Video/SRT paths via JSON
  3. Process management: Subprocess invocation

See INTEGRATION_GUIDE.md for complete code examples including:
  - Adding "Dub" button to UI
  - Configuration dialog
  - Progress monitoring
  - Integration with Qt/PySide6

USAGE EXAMPLES
==============

CLI Mode:
  madrac-dubbing.exe dub-cmd \
    --video video.mp4 \
    --srt subs.srt \
    --output dubbed.mkv \
    --language es \
    --voice female

API Mode:
  madrac-dubbing.exe api --port 5000
  # Then submit jobs via HTTP POST
